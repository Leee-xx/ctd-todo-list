# CTD To-Do List App

This repo contains the code for the CTD to-do list app homework, built using
React. It uses React Router to protect routes that require user authentication,
and Bootstrap to style UI elements.

## Setup
### Prerequisites
* [NodeJS](https://nodejs.org/en/download)
* NPM

After cloning this repo, you can install dependencies by doing the following in a terminal:

```bash
npm i
```

## Development
```bash
npm run dev
```

## Design Decisions
I used Bootstrap because of its popularity as a styling library. I prefer a
minimalist design aesthetic, and used color mainly as a way to indicate the
position of the cursor and where in the app a user is.
